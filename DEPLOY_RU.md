# Как запустить Ritual PumpPad

Это пошаговая инструкция для первого деплоя. Приватный ключ и seed phrase никому
не отправляй и никуда не вставляй, кроме своего кошелька.

## 1. Подготовь кошелек

1. Открой MetaMask или другой EVM-кошелек.
2. Добавь Ritual testnet:
   - Network name: `Ritual Chain Testnet`
   - RPC: `https://rpc.ritualfoundation.org`
   - Chain ID: `1979`
   - Currency: `RITUAL`
   - Explorer: `https://explorer.ritualfoundation.org`
3. Получи тестовые токены на faucet: `https://faucet.ritualfoundation.org`.

## 2. Задеплой контракт через Remix

1. Открой `https://remix.ethereum.org`.
2. Создай файл `RitualPumpPad.sol`.
3. Скопируй туда код из `contracts/RitualPumpPad.sol`.
4. Открой вкладку Solidity Compiler.
5. Выбери компилятор `0.8.24` или новее.
6. Нажми Compile.
7. Открой вкладку Deploy & Run Transactions.
8. В Environment выбери `Injected Provider - MetaMask`.
9. Убедись, что MetaMask находится в Ritual testnet.
10. В списке контрактов выбери `RitualPumpPad`.
11. В поле конструктора `treasury_` вставь свой адрес кошелька.
12. Нажми Deploy и подтверди транзакцию.
13. Скопируй адрес задеплоенного контракта.

## 3. Подключи контракт к сайту

Для быстрой проверки:

1. Открой `index.html`.
2. Найди блок Founder setup.
3. Вставь адрес контракта.
4. Нажми Save.
5. Подключи кошелек и попробуй создать токен.

Для публичного сайта:

1. Открой `src/config.js`.
2. Вставь адрес контракта:

```js
window.RITUAL_PUMPPAD_CONFIG = {
  factoryAddress: "0xYourDeployedFactory",
};
```

3. Сохрани файл.

## 4. Задеплой сайт на Vercel

Самый простой путь:

1. Создай GitHub-репозиторий.
2. Загрузи туда этот проект.
3. Открой `https://vercel.com/new`.
4. Выбери свой репозиторий.
5. Нажми Deploy.

Vercel должен определить проект как обычный статический сайт. Специальная сборка
не нужна.

## 5. Включи загрузку картинок

На сайте уже есть выбор картинки с компьютера или телефона. Чтобы картинки были
видны всем пользователям, а не только локально, нужно включить IPFS-загрузку
через Pinata:

1. Создай аккаунт на `https://pinata.cloud`.
2. Создай API key/JWT.
3. Открой настройки проекта в Vercel.
4. Добавь Environment Variable:

```text
PINATA_JWT=твой_pinata_jwt
```

5. Сделай Redeploy проекта.

После этого сайт будет сжимать картинку в браузере, отправлять ее в `/api/upload`,
пинить в IPFS через Pinata и записывать ссылку картинки в токен.

## 6. Проверка после деплоя

1. Открой сайт на Vercel.
2. Подключи кошелек.
3. Убедись, что кошелек переключился на Ritual testnet.
4. Создай тестовый токен с картинкой.
5. Купи немного токенов.
6. Попробуй продать часть обратно.

Это тестнет-приложение, не audited-контракт для реальных денег.
